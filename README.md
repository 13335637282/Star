# 如何使用 Star
1. 下载 JACOB.1.18 的 dll文件，首次打开程序时，应该会自动下载到 Star 目录下
2. 配置 config.json 将 lang, theme 调整到自己的喜好
3. (如果不在意音效可以不做) 检查 click_button.wav 是否已经在Star/sound/ui目录下，如果首次打开没有下载请手动下载
4. 配置完成后运行 `java -jar star.jar` 即可

## 常见问题
### 程序无法正常运行
请按照步骤检查：
   - 检查运行环境是否有 Java 23 及以上(不保证Java 23以下能正常运行，也不保证不能正常运行，这只是项目的构建版本)
   - 检查是否是最新的版本
   - 检查是否是 Windows 系统 (对不起，我们目前只支持Windows系统，未对其他系统做过测试)
   - 如果以上都是正常的，请提交log日志，提交issue
